jquery.scale demo

This is a demo for the scale jQuery plugin which resizes objects inside thier parent while maintaining the aspect ratio.

"fit" mode

$('obj').scale();

In this mode, the plugin will scale the object to the largest size possible while still fitting inside its parent and maintaining the aspect ratio.
