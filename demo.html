jquery.scale

The jQuery "scale" plugin resizes objects to fit inside thier parents while maintaining the aspect ratio. To use it, simply apply the plugin to any object, i.e.

$("#object").scale()

Optionally, one can also apply the "center" option, which will center the object horizontally and vertically inside the parent, i.e.

$("#object").scale("center")
